# Встановлення

Встановлення складається з декількох етапів:
- Встановлення мовного файлу.
- Встановлення модифікованої дати гри.
- Зміна мови у конфігурації гри.

Модифікація дати гри необхідна для заміни звичайного шрифту гри, на той що має підтримку української абетки. Для заміни я використовував шрифт Bitter-Regular, так як стандартний шрифт гри дуже кастрований, та не має підтримки майже усього. Це також означає що ви вільно зможете використовувати українську мову у чаті гри. А також українську латинку, для зацікавлених, так як підтримка діакритиків у зміненому шрифті теж присутня. 

Заванатжити всі потрібні файли ви можете [тут](https://drive.google.com/file/d/1CvmVzQmXdESWrS3PZVBhCuQ5XzA8pK1z/view?usp=sharing)

## Встановлення мовного файлу і модифікованої дати гри

Завантаживши архів з усіма потрібними файлами, все що від вас вимагається - це розпакувати його у вашу теку з грою, та замінити усі файли. Не змінюйте структуру видобутих файлів, просто розпакуйте і замініть ті файли, які про це питають.

## Зміна мови у конфігурації гри

У папці з грою також є тека `config`, у ній лежить файл `config.ini'. Вам необхідно вікдрити його через будь який доступний редактор тексту.
Всередені файлу ви знайдете
```
language=en
```
Ви маєте замінити `en` на `cs`.
```
language=cs
```

Та після цього все що вам залишається це зберегти цей файл та насолоджуватись українізованою грою.
Переклад був зроблений на базі "чеської мови", точніше, для гри він сприймається як чеська. Це пов'язано з тим, що Vagante по якійсь причині має обмежений перелік мов на які він взагалі може бути перекладений, тож я обрав саме чеську як оптимальний варіант.
